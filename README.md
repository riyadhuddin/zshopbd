# zshopbd

A food delivery platform connecting various foodies and restaurants in colaborative relatioship.

## Feature Category
### Three Main App Feature
** Pick and Parcel
** Seat booking
** Nearby Delivery aka freelance delivery

## Restaurant Feature
* Create Restaurant 
                - OwnerShip
* Manage Order
* Food Menu
      - Add
      - Delete
      - Edit
      - Coupon
## Admin
      * Price
      * Vat/Tax
      * Quality
